Model Overview

This model predicts whether a person's income is likely to be above 50K or not based on demographic and work-related details.

What it does

The model takes user inputs like age, education, workclass, occupation, hours per week, capital gain, capital loss, and a few other details.

After that, it returns one of two outputs:

0 = <=50K

1 = >50K

Model used

Algorithm XGBoost Classifier
Type Binary Classification
Dataset Adult Income Dataset

How it performed

Accuracy shows the overall correct predictions.

Precision shows how often the positive prediction was correct.

Recall shows how many real positive cases the model was able to catch.

F1 Score gives the balance between precision and recall.

Limitations

This is a prediction model, not a guarantee.

It should be used as support, not as final truth.
